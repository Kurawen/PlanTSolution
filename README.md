# PlanT
Корпоративное приложение для заметок

## Запуск
### Клонирование проекта 
```C
git clone https://github.com/Kurawen/PlanTSolution.git
```
### Backend
```bash
cd PlanTSolution/App && dotnet run
```
### Frontend
```bash
cd PlanTSolution/PlanT && npm run dev
```
